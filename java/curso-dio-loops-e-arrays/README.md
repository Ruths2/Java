## ➿ Estrutura de repetições (loops)

### 🔁 While
🔸Realiza uma repetição com o teste no **início**.

### 🔁 Do/While
🔸 Ao contrário do While o Do/While realiza uma repetição com o teste no **final**.

### 🔁 For
🔸Realiza uma repetição contada.
